# Swiggy Clone — Frontend Architecture UI

A pixel-perfect, highly accurate frontend desktop landing page clone of the Swiggy platform. This project was developed from scratch to master structural layout mechanics, component isolation, and advanced layout positioning using pure semantic HTML5 and CSS3.

## 🛠️ Technical Breakdown & Skills Demonstrated

### 1. Advanced Positioning & Component Layouts
* **UI Accuracy:** Translated complex UI elements from production designs into static layouts, focusing on precise margins, specific padding constraints, and proper typographic hierarchies.
* **Absolute Overlay Positioning:** Implemented contextual `position: absolute` styling for complex UI elements, including the floating `"PROMOTED"` badges pinned over restaurant thumbnails without disrupting structural flow.
* **Pseudo-elements (`::before`/`::after`):** Utilized advanced CSS pseudo-elements for decorative and layout structural elements (such as geometric UI markers and custom content formatting).

### 2. Semantic Document Object Model (DOM) Structure
* **Modern Semantics:** Structured the application with modern, readable HTML5 tags (`<header>`, `<nav>`, `<main>`, `<footer>`) ensuring excellent document hierarchy.
* **Maintainable Class Architectures:** Organized layout structures with structured CSS classes (e.g., parent-child selectors like `.place-name .name` and `.status`), mirroring professional styling habits.

### 3. Layout Engineering & Hierarchy
* **Dynamic Grid Alignments:** Engineered a multi-column restaurant grid featuring uniform card constraints, optimized aspect-ratio item blocks, and elegant hover-state scaling effects.
* **Component Optimization:** Organized the platform's multi-column footer layout with proper division of information into independent, scannable columns (`Company`, `Contact`, `Legal`).

## 🚀 Key Features

* **Authentic Header Navigation:** Complete global navigation layout housing location triggers, contextual user search, offer modules, support links, profile gateways, and interactive cart positioning.
* **Refined Filter Controls:** Accurate rendering of horizontal layout filters representing real-world query states (`Relevance`, `Delivery Time`, `Rating`, `Cost`).
* **Visual Identity & Assets:** Perfectly integrated platform assets, including SVG icons, custom promotional tags, and crisp application marketplace badges.
